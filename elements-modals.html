    Exit Modals
    Now Supported in Foundry!

    You can set a modal to show when the user's mouse leaves a particular element.

    • Attract attention when the user is about to leave your page by specifying "body".
    • Attract attention and offer discounts when a user leaves the "checkout" button.
    • Show a modal when a user scrolls past a particular section on your page.

    Exit modals can be cookies enabled so the modal can be restricted to showing only once.
